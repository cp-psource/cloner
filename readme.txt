Cloner kopiert jede ClassicPress-Webseite in Deinem Multisite-Netzwerk und verschiebt sie mit einem Klick auf eine beliebige URL im selben Netzwerk.

Cloner copies any ClassicPress website in your multisite network and moves it to any URL in the same network with one click.

== Description ==

= Deutsch =

Designänderungen vorzunehmen, eine neue Codezeile hinzuzufügen oder Dein Theme auf einer Live-Site zu optimieren, ist ein riskantes Geschäft. 
Nur das Aktivieren des falschen Plugins kann Deine Webseite beschädigen. 
Mit Cloner kannst Du Webseiten problemlos in Deinem Netzwerk verwalten und verschieben, um sie stressfrei bearbeiten zu können.

Migrieren, Stage, Backup
Mit Cloner können Webseiten-Migration, Backups sowie sicheres Staging und Bereitstellen problemlos durchgeführt werden. 
Verwende die Standardeinstellungen und kopiere Deine gesamte Webseite oder wähle nur die Elemente aus, die Du übertragen möchtest. 
Perfekt für die Verwaltung von Live-Webseiten.

= Englisch = 

Making design changes, adding a new line of code, or tweaking your theme on a live site is a risky business.
Just activating the wrong plugin can break your website.
With Cloner, you can easily manage and move websites around your network for hassle-free editing.

Migrate, Stage, Backup
Website migration, backups, and secure staging and deployment can be easily performed with Cloner.
Use the default settings and copy your entire website or just select the items you want to transfer.
Perfect for managing live websites.
